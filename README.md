# My Tasks – Task Management App

A simple and responsive task management application built using React.js. Users can create tasks with different tags and filter tasks based on selected categories.



## 📂 GitHub Repository

[GitHub Repository](https://github.com/GreeshmanthReddyBommireddy1/my_tasks)

---

## 📌 Features

- Add new tasks dynamically
- Assign tags to tasks
- Filter tasks based on tags
- Responsive UI design
- Simple and clean user interface
- Real-time task rendering using React state management

---

## 🛠️ Technologies Used

- React.js
- JavaScript
- HTML5
- CSS3

---

## 📸 Preview

The application allows users to:

- Create tasks using the input field
- Select tags from dropdown
- View added tasks instantly
- Filter tasks by clicking tag buttons

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/GreeshmanthReddyBommireddy1/my_tasks.git
```

### 2. Navigate to the project folder

```bash
cd my_tasks
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm start
```
